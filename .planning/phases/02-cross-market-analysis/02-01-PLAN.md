<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Export filename → Browser download | User-provided filename must be sanitized before triggering download (path traversal, XSS prevention) |
| Indicator data → Export file | Data from APIs may contain formula-like prefixes (=, +, -) that trigger CSV/Excel formula injection |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-01 | Tampering | export-csv.ts, export-xlsx.ts | mitigate | Sanitize filename with regex (alphanumeric + underscore/hyphen only), reject path characters (/, \, ..) |
| T-02-02 | Tampering | export-csv.ts, export-xlsx.ts | mitigate | Strip formula prefixes (=, +, -, @) from all exported string fields per RESEARCH.md Security Domain |
| T-02-03 | Information Disclosure | MultiSeriesChart.tsx | accept | Chart tooltip shows data values (no secrets), low-value target |
| T-02-04 | Denial of Service | data-alignment.ts | accept | Timestamp alignment is client-side, no external service dependency |
</threat_model>
