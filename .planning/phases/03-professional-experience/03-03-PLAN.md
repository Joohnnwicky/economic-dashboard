# FOMC detection algorithm (from RESEARCH.md)
1. Fetch DFEDTARU historical data
2. Iterate through data points
3. Compare current value to previous value
4. If prev !== curr: mark as FOMC event
5. Determine decision: curr > prev = 加息, curr < prev = 降息, curr === prev = 维持
6. Filter to past 1 year (D-14)

   export function detectFOMCMeetings(historical: HistoricalDataPoint[]): FOMCEvent[]

3. Detection algorithm:
   - Iterate from i=1 to historical.length
   - Compare historical[i-1].value to historical[i].value
   - Skip if prev === null or curr === null
   - If prev !== curr (rate changed):
     - decision: curr > prev ? '加息' : curr < prev ? '降息' : '维持'
     - color: DARK_THEME.accent[2] (hike) | DARK_THEME.accent[1] (cut) | DARK_THEME.textMuted (hold)
     - Note: DARK_THEME.accent[2] = '#f85149' (red), accent[1] = '#3fb950' (green), textMuted = '#8b949e' (gray)
   - Create FOMCEvent with timestamp, rate, decision, color

4. Historical range filter (D-14):
   - Calculate oneYearAgo: new Date(now.getFullYear() - 1, ...)
   - Filter events where timestamp >= oneYearAgo

5. Return FOMCEvent array

Utility is pure function, no API calls, no external dependencies beyond types.
